# Personal Website - Tufte Style

This is a personal website for Joseph Hosanna Sakaya, designed in the style of Edward Tufte's principles of information design. The website features clean typography, elegant margins, and a focus on content presentation.

## Features

- Clean, minimalist design
- Tufte-style typography and layout
- Responsive design for all devices
- Side notes and margin notes support
- Semantic HTML5 structure

## Usage

Simply open `index.html` in a web browser to view the website. The site is static and doesn't require any server-side processing.

## Customization

To customize the website:

1. Edit `index.html` to modify the content
2. Adjust styles in `tufte.css` to change the appearance
3. Add additional sections or pages as needed

## Design Philosophy

The website follows Edward Tufte's design principles:
- Typography-centric design
- Generous use of whitespace
- Margin notes for supplementary information
- High content-to-interface ratio

## Browser Support

The website is compatible with all modern browsers and is responsive across different screen sizes.
